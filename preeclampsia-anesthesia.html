Anesthesia for Obstetrics

重度子癇前症診斷標準

符合以下任一項即為重度子癇前症：

嚴重高血壓： - SBP ≥160 mmHg 或 DBP ≥110 mmHg，相距 ≥4 小時測量兩次（臥床中） - 確認後應立即開始降壓治療，無需等待 4 小時

中樞神經系統症狀： - 新發腦/視覺障礙：photopsia、scotomata、皮質盲、視網膜血管痙攣 - 嚴重頭痛（最嚴重頭痛）或持續加重、對 acetaminophen 無效

肝功能異常： - 血清轉氨酶 > 2× ULN，或嚴重右上腹/上腹痛無其他病因

血小板減少： - 血小板 < 100,000/μL

腎功能損傷： - 血清肌酐 > 1.1 mg/dL 或在無其他腎病情況下加倍

肺水腫

❗ 重要

重度子癇前症的降壓目標：SBP 140–150 mmHg、DBP 90–100 mmHg。過度降壓會影響胎盤灌流。首選藥物：labetalol IV、hydralazine IV 或 nifedipine PO。

子宮血流生理

  • 子宮血流幾乎沒有自動調節；血管在妊娠期幾乎完全擴張
  • 子宮及胎盤血流直接依賴母體心輸出量
  • 子宮灌流壓 = 子宮動脈壓 − 子宮靜脈壓
  • 降低子宮灌流的原因：母體低血壓（低血容量、全身/椎管內麻醉後血管舒張）、主動脈腔靜脈壓迫（aortocaval compression）
🚫 注意事項

妊娠 20 週後仰臥位可造成下腔靜脈壓迫 → 靜脈回流減少 → 心輸出量下降 → 子宮灌流減少。應常規左側臥（左傾 15–30°）或子宮左移。

剖腹產的全身麻醉 vs 椎管內麻醉

  • 1979–1990 年資料：GA 的母體死亡率是 neuraxial 的 16.7 倍（RR 16.7）
  • 1997–2002 年資料：RR 下降至 1.7（95% CI 0.6–4.6，無統計顯著差異）
  • 現代 GA 技術進步，風險差距顯著縮小

產程鎮痛（Neuraxial）的皮節覆蓋

  • 第一產程：T10–L1（子宮收縮、宮頸擴張的疼痛）
  • 第二產程：擴展至 S2–S4（會陰部疼痛）

胎兒手術的麻醉管理

  • 維持 ETCO₂ 28–32 mmHg（eucapnia for fetus）
  • 靜脈輸液限制 < 2 L（防止 tocolytic 相關肺水腫）
  • 使用 nitroglycerin 大劑量時也需限制補液

Beta-2 Agonist 安胎藥作用機轉

  • β₂ receptor → ↑ adenyl cyclase → ↑ cAMP → protein kinase 活化 → 細胞內鈣降低 → 抑制 myosin light-chain kinase → 子宮平滑肌舒張
  • 長期使用 → receptor 脫敏（tachyphylaxis）→ 療效下降
