Critical Care Anesthesiology

ℹ️ 背景

「Medical（pharmacological）circulatory support」指以血管活性與強心藥物維持灌流，與機械循環支持（IABP、Impella、ECMO）並列為 cardiogenic shock（CS）的處置主軸。選藥取決於目標：提升 SVR（升壓）、提升 CO（正性肌力）或降低後負荷（血管舒張）。詳見 [[sinha-2025-cardiogenic-shock]]。

三大作用方向

flowchart TD
    A[循環衰竭 / 低灌流] --> B{主要缺陷？}
    B -->|SVR 過低| C[Vasoconstriction]
    B -->|CO 過低| D[Positive inotropy]
    B -->|後負荷過高| E[Vasodilation]
    C --> C1[Vasopressin / Phenylephrine / Norepinephrine]
    D --> D1[Epinephrine / Dopamine / Dobutamine / Milrinone / Levosimendan]
    E --> E1[Nitroglycerin / Nitroprusside]

受體機轉與血流動力學摘要

類別 代表藥 主要受體 淨血流動力學
Inopressor（升壓兼強心） Norepinephrine、Epinephrine、Dopamine α₁ + β₁ ↑SVR、↑BP、↑CO
Inodilator（強心兼舒張） Dobutamine、Milrinone β₁ / PDE-3 ↑CO、↓SVR（注意低血壓）
Vasopressor（純升壓） Phenylephrine、Vasopressin α₁ / V1 ↑SVR、↑BP、CO 不變或↓
Vasodilator（降後負荷） Nitroprusside、Nitroglycerin NO ↓SVR、↓BP
Chronotrope（增心率） Isoproterenol β₁β₂ ↑HR、↑CO、↓SVR
Inotrope（鈣敏化） Levosimendan troponin C ↑CO、↓SVR
❗ 重要

Dopamine 的劑量依賴效應：低劑量（2–5 µg/kg/min）以 D1 受體為主擴張腎／內臟血管；中劑量（5–10）β₁ 正性肌力；高劑量（10–20）α₁ 升壓。臨床上 CS 已不再常規使用「腎劑量 dopamine」保腎。

🚫 注意事項
  • Inodilator（dobutamine、milrinone） 增加 CO 但降低 SVR，對已低血壓的 CS 患者可能需併用升壓藥。
  • Milrinone 經腎排除、半衰期長（~2–4 h），腎功能不全者作用延長且不易滴定。
  • Vasopressin 收縮體循環但相對不增加肺血管阻力，常用於右心衰竭／肺高壓情境（見 [[pulmonary-hypertension-anesthesia]]）。
  • Levosimendan 鈣敏化不增加心肌耗氧，但會降低 SVR，常見低血壓。
