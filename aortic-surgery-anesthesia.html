Anesthesia for Vascular Surgery

Crawford 胸腹主動脈瘤分類（Crawford Classification）

Crawford 分類依據動脈瘤累及的主動脈範圍：

類型 範圍 脊髓缺血風險
Type I 降主動脈（descending）上段至腹主動脈上段（不含腎動脈以下） 中等
Type II 降主動脈全段至腎動脈以下（最廣泛） 最高（15–40%）
Type III 降主動脈下段至腎動脈以下 中等
Type IV 全段腹主動脈（腎動脈以下至雙側髂動脈） 最低
Type V 降主動脈下段至腎動脈上段（有作者加入） 中等
❗ 重要

Type II TAAA 手術的脊髓缺血（截癱）風險最高，因為需要阻斷最多的脊髓供血段（包括 Adamkiewicz 動脈，T8–L1 段）。

脊髓保護策略

  1. 脊髓液引流（CSF drainage）：降低 CSF 壓力（目標 < 10 mmHg）→ 提高脊髓灌流壓（CPP = MAP − CSF pressure）
  2. 遠端主動脈灌流（Distal aortic perfusion）：維持 distal aortic pressure > 60 mmHg
  3. 低體溫（Hypothermia）：降低脊髓代謝需求
  4. Evoked potential 監測：SSEP 和 MEP（tc-MEP）
  5. 再植入脊髓關鍵肋間/腰椎動脈（外科技術）
  6. 術後血壓管理：維持 MAP > 80–90 mmHg 至少 24–48 小時
💡 臨床珍珠

術中 SSEP/MEP 改變（振幅下降 >50% 或潛伏期延長 >10%）應立即通知外科醫師，考慮再植入肋間動脈或增加遠端灌流壓。

術中監測

  • 動脈導管：雙側（右側橈動脈 + 股動脈，監測上下肢壓力差）
  • TEE：心功能、容積狀態
  • 肺動脈導管：視情況置入
  • 神經電生理監測（SSEP + MEP）
  • 核心體溫
  • 尿量

腎臟保護策略

  • 冷藏腎動脈灌流（Cold renal perfusion）
  • 輕度低體溫
  • 維持血壓 → 腎臟灌流壓
  • Fenoldopam（多巴胺 D₁ 激動劑）：增加腎血流，術中腎保護尚有爭議

出血處置

  • 大量輸血方案（MTP）準備
  • Cell Saver
  • 術前備 FFP、血小板、cryoprecipitate
  • 術後凝血功能監測（TEG/ROTEM 指引）
